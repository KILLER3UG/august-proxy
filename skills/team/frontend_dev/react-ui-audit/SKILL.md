---
name: react-ui-audit
description: Audit React UI changes for layout, accessibility, interaction, and browser behavior
trigger: react ui audit, frontend review, component behavior check
---
Use this skill when reviewing frontend UI work.

- Inspect the affected React/Vite/Tailwind files first.
- Check layout, responsive behavior, accessibility labels, keyboard interaction, loading/error states, and browser console errors.
- Prefer evidence from the DOM, tests, screenshots, or browser console over assumptions.
- Report concrete findings with file paths and reproducible steps.
