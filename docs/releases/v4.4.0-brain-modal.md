# v4.4.0-brain-modal вҖ” Titlebar Brain Icon вҶ’ Mini Modal

> **Released:** 2026-06-30
> **Spec gap closed:** В§12 вҖ” the Brain dashboard wasn't reachable in the current chat-first UI. The nav items existed but no surface rendered them.
> **Previous release:** [v4.3.0-brain-activity](./v4.3.0-brain-activity.md)

## What's new

The user asked: *"where do i find the /brain and activity tab?"* and the honest answer was *nowhere in the UI вҖ” only via Command Palette or direct URL*. This release closes that gap with the smallest possible change:

- A **рҹ§  Brain icon in the title bar** (next to Settings)
- Clicking it opens a **small modal that defaults to the Activity tab** вҖ” the realtime brain flow the user actually wants to see
- Three tabs inside: Activity / Learning / System Health (reuses the existing components from v3+v4.3)
- Esc, Г— button, and backdrop click all close

### Before
- `/brain` route registered but unreachable from any visual surface in the chat shell
- Command Palette (`Cmd+K` / `Ctrl+P`) was the only nav path

### After
- Titlebar `рҹ§ ` icon вҶ’ click вҶ’ modal opens, Activity tab visible, events streaming in via SSE
