## What's fixed

### Critical bugs (chat was broken without these)

| Commit | Fix |
|--------|-----|
| `fbfc2ad` | `build_system_prompt` now accepts `cached_t12` keyword argument (Phase 7 cache hook). Without this, every chat turn raised `TypeError`. |
| `9b42fad` | `auto_memories` table has new `updated_at` column with idempotent migration. Without this, any duplicate-key save raised `OperationalError`, breaking background review on its second cycle. |
| `327057d` | `<failure_feedback>` Tier 3 block now has a producer with 3-turn decay. Without this, tool errors fell through as raw strings in chat tool results (panic-inducing tracebacks). |
