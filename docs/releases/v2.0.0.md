# v2.0.0 — Phases 8-10 Bring-up

**Date:** 2026-06-29
**Tag:** v2.0.0 (local — not pushed)

## What ships

### Foundation
- **Model fleet** (`app/services/workbench/model_fleet.py`) — Cortex / Cerebellum / Hippocampus / Prefrontal role mapping with config override
- **Scheduler** (`app/services/scheduler.py` v2 class) — Centralized periodic + idle-triggered tasks

### Phase 8 — Subconscious daemons
- Daemons actually call Cerebellum via `model_fleet.get_model_for_role("cerebellum")`
- Tool allowlist enforced at dispatch layer (mutating commands blocked: `rm`, `mv`, `del`, `format`, `mkfs`, `dd`, `shutdown`, `chown`, etc.)
- Watch conditions work: `on_completion`, `on_match:KEYWORD` (case-insensitive), `on_change` (md5 hash)
- Max 3 concurrent daemons per session; result expiry after 5 turns; exponential backoff
- `[CRITICAL]` prefix preserved through `<subconscious_updates>` Tier 3

### Phase 9 — Autonomous cognitive maintenance
- **9a Consolidation** uses Hippocampus LLM to merge duplicates, promote patterns to facts, delete stale
- Recent 20 rules protected from deletion (safety)
- All writes go through `db_writer.enqueue_write`
- **9b Delta engine** infers rules via Hippocampus (batched diffs)
- Delta engine subscribes to env-watcher events
- **9c Timeline** populates on session end + hourly sweep catches missed sessions

### Phase 10 — Advanced frontiers
- **10.1 Blackboard** adaptive TTL (`max(poll_interval*2, 60s)`)
- `ack=True` parameter on `read_blackboard` deletes the note on read
- `<blackboard_state>` injected in Tier 3
- **10.2 Environment watcher** uses `watchdog` with ignore patterns (`.pyc`, `node_modules`, `.git/objects`, swap files, logs)
- Rate limit (1 update/2s); `<environment>` injected in Tier 3
- **10.3 Verifier reflex** injects `<verifier_gate>` on `phase=review|complete`
- Specific command rendered when supplied, generic reminder otherwise
- **10.4 Skill genesis** drafts SKILL.md via Prefrontal; `pending_skills` table
- User approval flow: `approve_pending_skill` moves to active, `reject_pending_skill` cleans up
- Rate-limited (1 auto-gen skill per day)
